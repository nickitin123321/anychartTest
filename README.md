# Установка

Требуется ввести команды:
- Установка пакетов:
 npm install
- Запуск сервера:
npm run server

# Список используемых технологий

- Node.js (запуск сервера)
- Java script  (контроль данных, отрисовка графика на svg)
- Html (разметка страницы, поле ввода, кнопка)
- CSS (стили графика)
